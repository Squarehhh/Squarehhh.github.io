BlueMoon

前期准备

靶机地址：https://www.vulnhub.com/entry/bluemoon-2021,679/

攻击机IP：192.168.119.132

靶机IP：192.168.119.143

信息收集及利用

nmap扫

开放了21,22,80端口

21端口

无法匿名登录

80端口

扫目录

之前一直用的dirsearch，看见别人用了gobuster，尝试用了下发现好像效率高很多？

看下/hidden_text

有个二维码，找个在线的扫一下看看

发现了一组账号密码，尝试ftp登录

成功登录

看见了两个txt文件，下载下来看看

robin可能是用户名，下面的应该是密码本，加上开放了22端口，猜测为爆破ssh连接

开始爆破

得到密码k4rv3ndh4nh4ck3r

ssh登录

成功登录咯

提权

查看下文件

查看user1

查看权限

发现可以利用Jerry账户的feedback.sh，查看一下

使用jerry的身份运行脚本sudo -u jerry /home/robin/project/feedback.sh

升级shell

python3 -c 'import pty;pty.spawn("/bin/bash")';

查看文件，查看user2.txt

接下来就是提权到root，上面查看id发现是docker组的

尝试docker提权

sudo docker run -v /:/mnt --rm -it alpine chroot /mnt sh

成功提权

升级shell

查看最后一个flag

完成！！

探测不到靶机IP解决方式

设置为nat网络模式

如果还是不行，就重启，长按shift

选择第二个，按e进入编辑

将ro改为rw single init=/bin/bash，按住ctrl+x重启服务

这里我替换过了，在原本的里面ens33的地方为其他的内容，靶机网卡配置文件 与 实际网卡名称 并不一致，因此网卡无法正常启动

重启就行

学习了一下ftp的一个攻击利用方式，也了解到了gobuster工具，扩展了工具库。之前在WAYNE MANOR靶场没有做到的升级shell和提权，在这里也学到了，从robin到jerry用户，然后由在docker用户组，进行docker提权，最终成功拿下root权限。总的来说学到了很多东西。
